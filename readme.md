# SAP CAP Bookshop Case Study

This repository contains my solution for the SAP Development case study.

The project is based on the SAP Cloud Application Programming Model (CAP) sample application created from the official getting started guide. I extended the standard template by enabling PostgreSQL persistence, adding a `rating` field (1–10), and introducing additional entities such as **Users**, **Orders** and **OrderItems**.

## Overview

The solution includes:

- A CAP backend based on the sample bookshop project
- PostgreSQL persistence using Docker
- Extended `Books` entity with a `rating` field
- Additional entities: **Users**, **Orders** and **OrderItems**.
- Validation to ensure ratings stay between 1 and 10
- React Custom UI to consume CAP API's

## Installation

Cloning Repository

```bash
git clone https://github.com/Nkululeko1999/bookshop.git
cd bookshop
npm install
```

Install PostgreSQL adapter:
```bash
npm add @cap-js/postgres
```

---

## Configuration

Create .cdsrc.json in the root folder

```bash
{
  "requires": {
    "db": {
      "[pg]": {
        "kind": "postgres",
        "credentials": {
          "host": "localhost", "port": 5432,
          "user": "postgres",
          "password": <password>,
          "database": "bookshop"
        }
      }
    }
  }
}
```

## Data Model Extension

Changed Integer ID Keys to UUID - autogenerated.

```bash
using {
  Currency,
  managed,
  sap
} from '@sap/cds/common';

namespace sap.capire.bookshop;

entity Users : managed {
  key ID    : UUID;

      names  : String;
      email : String;
      password : String;
      avatar : String;
      avatarID : String;
}

entity Books : managed {
  key ID       : UUID;
      title    : localized String;
      descr    : localized String;
      author   : Association to Authors;
      genre    : Association to Genres;
      stock    : Integer;
      price    : Decimal;
      rating   : Integer;
      type     : String;
      pages    : Integer;
      ISBN     : String;
      imageUrl : String;
      imageID  : String;
      currency : Currency;
}

entity Authors : managed {
  key ID       : UUID;
      name     : String;
      avatar   : String;
      avatarID : String;
      books    : Association to many Books
                   on books.author = $self;
}

entity Genres : sap.common.CodeList {
  key ID     : UUID;
      parent : Association to Genres;
}

entity Orders : managed {
  key ID    : UUID;

      user  : Association to Users;
      total : Decimal;

      items : Composition of many OrderItems
                on items.order = $self;
}

entity OrderItems : managed {
  key ID       : UUID;

      book     : Association to Books;
      quantity : Integer;
      price    : Decimal;

      order    : Association to Orders;
}
```

Additional entities:
- Users
- Orders
- OrderItems
